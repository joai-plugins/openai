# OpenAI

Connect OpenAI to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/openai) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-openai@joai-claude-plugins
```

### Codex

Install this app from the official JoAi Codex marketplace in Codex.

`joai-openai`

If your Codex build only supports MCP registration, use the MCP endpoint below as a manual fallback.

### Cursor

Use the [Cursor install link](https://cursor.com/en-US/install-mcp?name=joai-openai&config=eyJ1cmwiOiJodHRwczovL2NvcnRleC5qb2FpLmFpL21jcC9hcHBzL29wZW5haSJ9) to install directly.

Or copy this fallback URL:

```
https://cortex.joai.ai/mcp/apps/openai
```


### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/openai
```
