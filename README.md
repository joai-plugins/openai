# OpenAI

Connect OpenAI to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/openai) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-openai@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-openai --url https://cortex.joai.ai/mcp/apps/openai
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/openai.git ~/.cursor/plugins/local/joai-openai
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-openai&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktb3BlbmFpIjp7InVybCI6Imh0dHBzOi8vY29ydGV4LmpvYWkuYWkvbWNwL2FwcHMvb3BlbmFpIn19fQ==) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/openai
```
